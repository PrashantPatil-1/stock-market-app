## Stock Market Application

This project is a Stock Market Application developed using Spring Boot for the backend and ReactJS for the frontend. The application allows users to perform CRUD operations on stock market data and provides authentication features.

## Project Structure:

Backend: Spring Boot
Frontend: ReactJS
Database: MySQL
Tools & Libraries:
Spring Security for authentication
JdbcTemplate for database operations
Log4j for logging
Bootstrap for frontend styling


## Features
User Registration and Login
CRUD operations for stock market data
Stock search functionality
Secure password storage using BCryptPasswordEncoder

## Execution:
backend:
Open in STS
Import the Project
Configure Database
Select Run As > Spring Boot App.



frontend : 
cd stockmarket-frontend
npm install - command to install the required dependencies
npm start -Start the development server by running
